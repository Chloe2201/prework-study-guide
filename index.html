HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content to show the user.
  • It is preferrable to use semantic HTML to ensure your code is accessible to all users and optimise SEO.
  • HTML is the foundation of website creation.
  • The meta data is located in the head element and will not render on the website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
  • The most common way of using CSS is to use external style sheets.
  • CSS adds colours, shapes and makes our content more visually appealing.
  • * as a selector will apply that rule to all elements visible on the page.

Git

  • Git is to be used via the terminal.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: pulls the latest main branch on the local environment.
  • git push origin branch-name: push the local changes to the remote Github branch.
